= Deploying an egress router pod in redirect mode
include::modules/common-attributes.adoc[]
:context: deploying-egress-router-cni-redirection

toc::[]

As a cluster administrator, you can deploy an egress router pod to redirect traffic to specified destination IP addresses from a reserved source IP address.

The egress router implementation uses an egress router Container Network Interface (CNI) plug-in. You can use this implementation with the OVN-Kubernetes network provider or the OpenShift SDN network provider.
